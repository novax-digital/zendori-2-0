# Brand & Design Assets

Ablage für Markenmaterial, das später ins Zendori-Design einfließt. **Noch keine
Design-Phase** — hier nur sammeln, wird zu gegebener Zeit umgesetzt.

Wohin womit:

- `logos/` — Logos (SVG bevorzugt, sonst hochauflösendes PNG), Favicon, App-Icon,
  hell/dunkel-Varianten
- `colors/` — Farbpalette / Brand-Farben (z. B. Hex-Liste, Farbdatei)
- `fonts/` — Schriftdateien oder Angabe der verwendeten Web-Fonts
- `guidelines/` — Brand-Guidelines / Styleguide (PDF o. Ä.), Tonalität, Do/Don't

Große Binärdateien einfach hier ablegen; die eigentliche Anwendung nutzt Assets
später aus `apps/web/public/`. Dieser Ordner ist die Quelle/Staging-Fläche, aus
der wir das UI-Theming (Farben pro Org, Logo, Widget-Branding) ableiten.
